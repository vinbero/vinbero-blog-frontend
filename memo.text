webpack을 이용하여 개발. require을 사용해 모듈을 로드할 수 있다. require.js는 오래된 기술
webpack을 사용하려면 nodejs도 설치해야 한다.
프로젝트 디렉터리에서
npm init
을 입력하면 package.json파일이 생성된다. 파이썬의 virtualenv처럼 모듈을 로컬에 설치할 경우, 설치된 모듈의 관련 정보가 저장되는 파일이다.
npm install -S webpack
을 입력한다 -S 옵션은 package.json 파일을 업데이트 하라는 뜻이다.
